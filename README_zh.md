# 鸣潮二周年团子赛跑模拟器

专为《鸣潮》二周年「团子赛跑」活动开发的网页版同人模拟器。

通过本项目，你可以自由组建 6 团子参赛阵容、深度推演对战细节，并利用蒙特卡洛算法进行海量数据模拟，从而评估各阵容的真实胜率与预期表现。

## 核心功能

- 交互式对战引擎：提供精确的比赛控制，支持逐帧单步推演、完整回合步进、自动播放以及一键生成结算结果。
- 完整锦标赛赛制：一比一复刻游戏内锦标赛流程，支持预赛模拟，并允许用户手动干预、自定义总决赛发车顺位。
- 蒙特卡洛数据分析：支持执行 100、1,000 或 10,000 次量级的批量模拟，直观输出阵容胜率、名次分布方差以及特定条件下的数据洞察。
- 体验优化：原生支持深色/浅色主题与中英双语切换，并通过本地存储自动记忆用户的阵容配置与界面偏好。

## 技术栈

本项目为纯前端单页应用 (SPA)。

- 核心框架：React 19
- 开发语言：TypeScript
- 构建工具：Vite
- 样式方案：Tailwind CSS

## 本地部署

请确保本地已安装 Node.js 20 或更高版本。

1. 安装项目依赖：
  npm install
2. 启动开发服务器：
  npm run dev
3. 构建生产环境代码：
  npm run build
4. 执行代码规范检查：
  npm run lint

## 目录结构

- /src/components: 界面 UI 组件与数据分析面板。
- /src/services: 核心模拟引擎，包含对战逻辑、锦标赛流程控制与蒙特卡洛算法实现。
- /src/constants: 静态配置数据，如棋盘规则与团子基础属性。
- /src/hooks: 封装 React 状态管理与本地持久化逻辑。
- /src/i18n: 本地化多语言字典与辅助函数。
